# 🛍️ E-commerce API

Modern RESTful API for e-commerce management, built with NestJS and Oracle Database.

## 🌟 Features

- 🔐 JWT Authentication
- 👥 User and role management
- 📁 Hierarchical category management
- 🌐 Multi-language support
- 📝 Complete audit
- 🔍 Optimized queries
